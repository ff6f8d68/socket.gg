Welcome to the Socket.GG Extension Wiki! This page provides detailed information on how to use the Socket.GG extension for Scratch.

Introduction

The Socket.GG extension allows you to connect to a signaling server, join rooms, send and receive messages and data streams, and handle events when messages or data are received in the current room.

Installation

Step 1: Set Up the Signaling Server

First, you need to set up a signaling server using Node.js and Socket.IO. Here's a simple example:


// server.js
const express = require('express');
const http = require('http');
const socketIo = require('socket.io');
const cors = require('cors');

const app = express();
const server = http.createServer(app);
const io = socketIo(server, {
    cors: {
        origin: '*',
        methods: ['GET', 'POST'],
        credentials: true
    }
});

let messages = {};
let privateMessages = {};

io.on('connection', (socket) => {
    console.log('A user connected:', socket.id);

    socket.on('joinRoom', (room) => {
        console.log(`User ${socket.id} joined room: ${room}`);
        socket.join(room);
        socket.currentRoom = room;
        io.to(room).emit('userJoined', { userId: socket.id });
        socket.emit('chatHistory', { room, messages: messages[room] || [] });
    });

    socket.on('leaveRoom', (room) => {
        console.log(`User ${socket.id} left room: ${room}`);
        socket.leave(room);
        delete socket.currentRoom;
        io.to(room).emit('userLeft', { userId: socket.id });
    });

    socket.on('message', (data) => {
        console.log('Received message:', data);
        if (!messages[data.room]) {
            messages[data.room] = [];
        }
        messages[data.room].push(data);
        io.to(data.room).emit('message', data);
    });

    socket.on('privateMessage', (data) => {
        console.log('Received private message:', data);
        io.to(data.toUserId).emit('privateMessage', data);
    });

    socket.on('fileTransfer', (data) => {
        console.log('Received file transfer request:', data);
        io.to(data.room).emit('fileTransferData', data);
    });

    socket.on('streamData', (data) => {
        console.log('Received stream data:', data);
        io.to(data.room).emit('streamData', data);
    });

    socket.on('disconnect', () => {
        console.log('User disconnected:', socket.id);
        if (socket.currentRoom) {
            io.to(socket.currentRoom).emit('userLeft', { userId: socket.id });
        }
        Object.keys(socket.rooms).forEach(room => {
            if (room !== socket.id) {
                console.log(`User ${socket.id} left room: ${room}`);
                socket.leave(room);
            }
        });
    });
});

server.listen(2415, () => {
    console.log('Signaling server is running on port 2415');
});

Save this code in a file named `server.js` and run it using Node.js:


node server.js

Step 2: Load the Extension in Scratch

To load this extension in the Scratch editor, follow these steps:

  1. Open the Scratch editor (either locally or online).
  2. Go to the "Extensions" tab.
  3. Click on "Add Extension" and select "Pen" or any other extension to open the extension manager.
  4. Click on "My Extensions" and then "Add Extension".
  5. Enter the URL of the hosted `extension.js` file: https://socket-gg.netlify.app/extension.js.
  6. Click "Import".

Usage

Blocks

Block Type Block Text Description
Command connect to server [SERVER_URL] Connects to the signaling server at the specified URL.
Command set user to [USER_ID] Sets the user identifier for the current session.
Command join room [ROOM] Joins the specified room on the server.
Command leave room [ROOM] Leaves the specified room on the server.
Command send message [MESSAGE] to current room Sends a message to the current room.
Command send data [DATA] to current room Sends data to the current room.
Reporter get current message Returns the most recent message received from the current room.
Reporter get all messages Returns all messages received from the current room.
Reporter get current data Returns the most recent data received from the current room.
Hat when message received in current room Triggers when a message is received in the current room.
Hat when data received in current room Triggers when data is received in the current room.

Example Project

Here’s a simple example of how you might use the blocks in a Scratch project:

  • Set User to ()
    • Use the "set user to [USER_ID]" block to set a user identifier, e.g., "User1".
  • Connect to Server
    • Use the "connect to server [SERVER_URL]" block to establish a connection to the server. For example, use http://localhost:3000 as the server URL.
  • Join Room
    • Use the "join room [ROOM]" block to join a specific room, e.g., "Room1".
  • Send Message to Current Room
    • Use the "send message [MESSAGE] to current room" block to send a message to the current room.
  • Send Data to Current Room
    • Use the "send data [DATA] to current room" block to send a data stream to the current room.
  • Get Current Message
    • Use the "get current message" block to retrieve the most recent message received from the server.
  • Get All Messages
    • Use the "get all messages" block to retrieve all messages received from the server.
  • Get Current Data
    • Use the "get current data" block to retrieve the most recent data stream received from the server.
  • Leave Room
    • Use the "leave room [ROOM]" block to leave a specific room.
  • Event-Driven Blocks
    • Use the "when message received in current room" hat block to execute scripts when a message is received in the current room.
    • Use the "when data received in current room" hat block to execute scripts when data is received in the current room.

Debugging Tips

  • Check Network Requests: Use the browser's developer tools (F12) to check network requests and ensure the Socket.IO library is being loaded correctly.
  • Console Logs: If you prefer to use console.log for debugging, you can open the browser's console to see detailed logs.

Example Script Using Event-Driven Blocks

Here’s an example of how you might use the event-driven blocks in a Scratch project:

  1. When Green Flag Clicked
    • Use the "when green flag clicked" block to start the project.
  2. Set User to ()
    • Use the "set user to [USER_ID]" block to set a user identifier, e.g., "User1".
  3. Connect to Server
    • Use the "connect to server [SERVER_URL]" block to establish a connection to the server. For example, use http://localhost:3000 as the server URL.
  4. Join Room
    • Use the "join room [ROOM]" block to join a specific room, e.g., "Room1".
  5. When Message Received in Current Room
    • Use the "when message received in current room" hat block.
    • Inside this hat block, use the "get current message" block to retrieve the message and perform actions based on the message.
  6. When Data Received in Current Room
    • Use the "when data received in current room" hat block.
    • Inside this hat block, use the "get current data" block to retrieve the data and perform actions based on the data.
